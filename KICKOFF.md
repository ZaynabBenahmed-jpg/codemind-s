## Présentation du projet

CodeMind-S est un Small Language Model (SLM) de moins
de 500 millions de paramètres fine-tuné sur un corpus
de code multi-langages (Python, JavaScript TypeScript
Java, C#, C, C++), capable de :
- Générer du code
- Expliquer du code existant
- Corriger des bugs


## Décisions prises

- Modèle retenu : CodeBERT (125M paramètres, licence MIT)
- Outil de gestion de projet : GitHub Issues
- Environnement d'entraînement : Google Colab T4 gratuit
- Stockage du modèle : HuggingFace Hub
- Monitoring : Weights & Biases
- Fréquence des réunions : 2 fois par semaine
